Scala

多范式语言，集成了面向对象和函数式编程的特性。

  • Scala 中一切值都是对象。对象的类型和行为由类（class）和特质（trait）来描述。

  • Scala 是一门函数式语言。Scala 为定义匿名函数提供了轻量级的语法，支持高阶函数、允许函数嵌套和柯里化（currying）。

  • Scala 是静态类型的。

  • Scala 是可扩展的。

  • Scala 与 Java 运行环境（JRE）可以进行良好的互操作。
